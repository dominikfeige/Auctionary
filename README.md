## Anleitung für Programmierer

#### Installation der Abhängigkeiten

```bash
# Server Abhängigkeiten
$ npm install

# Client Abhängigkeiten
$ cd frontend
$ npm install
```

#### Befehl für das starten des Servers

```bash
# Webserver starten
$ npm run start
```
